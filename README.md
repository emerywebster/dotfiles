Based on [Lars Kappert's dotfiles](https://github.com/webpro/dotfiles)

## Install

    sudo softwareupdate -i -a
    xcode-select --install

    git clone https://github.com/emerywebster/dotfiles.git
    source dotfiles/install.sh
